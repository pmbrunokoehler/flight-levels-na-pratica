# Prompts de IA por fase — Flight Levels nível 2

Biblioteca de prompts para apoiar cada fase de uma implementação de Flight Levels nível 2, do diagnóstico inicial à melhoria contínua. Os prompts são específicos de coordenação entre times; para prompts genéricos de gestão de projetos, veja o repositório [ai-pm-playbook](https://github.com/brunokoehler/ai-pm-playbook).

Esta pasta é a camada prática do [estudo de caso](../README.md): leia o caso para entender o que foi feito, use estes prompts para fazer no seu contexto.

## Navegação por fase

1. [Preparação e diagnóstico](01-preparacao-e-diagnostico.md) — antes de qualquer board: mapear o fluxo, decidir se FL2 serve, identificar flight items.
2. [Design e implementação](02-design-e-implementacao.md) — desenhar o board, calibrar itens, desenhar cadências.
3. [Operação e facilitação](03-operacao-e-facilitacao.md) — preparar cadências, destravar dependências, conectar OKR à entrega.
4. [Monitoramento e métricas](04-monitoramento-e-metricas.md) — interpretar fluxo, auditar saúde do sistema, comunicar à liderança.
5. [Melhoria contínua](05-melhoria-continua.md) — retro do sistema, detectar desvios, planejar expansão.

## Como usar

Cada prompt traz quando usar, o que você recebe, o texto pronto para copiar e uma dica. Preencha as variáveis entre `[colchetes]` com o contexto real e refine a resposta em diálogo. A IA acelera o raciocínio; a decisão é sempre sua.
